# 应用详情

本项目为 Go Web 框架 `iris` 的入门程序，部署到阿里云 Serverless 平台（函数计算 FC）。

Iris是一个高效且设计良好的跨平台Go Web框架，具有强大的功能集，支持 AWS Lambda、gRPC、MVC、Websockets、Sessions等功能，是构建高性能 Web 应用程序和 API 的不二之选。

</appdetail>
